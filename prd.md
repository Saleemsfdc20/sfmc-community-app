🚀 PRD — SFMC Mumbai Experience Hub
🧭 1. Product Overview

Product Name:
SFMC Mumbai Experience Hub

Type:
QR-based Event Check-in + Community Engagement Platform

Purpose:
To seamlessly capture attendee data, enhance engagement during events, and build a long-term community intelligence system.

🎯 2. Problem Statement

Current event tracking is:

Manual / scattered
No centralized attendee data
No engagement after check-in
No insights (first-time vs returning, interests, etc.)

👉 Result: Missed opportunities for growth and community building.

💡 3. Solution

Build a mobile-first web application where:

Users scan a QR code
Register instantly
Enter an interactive experience
Data gets stored and visualized in real-time

👉 Turning attendance into engagement + insights

👥 4. Target Users
Salesforce Marketers
Developers / Admins
Students / Freshers
First-time attendees
Returning community members
🔥 5. Key Features
🔹 5.1 QR-Based Check-In
Instant access via QR scan
No app install required
🔹 5.2 Smart Registration Form

Fields:

Name
Email
Mobile
City
Role
First-time attendee (Yes/No)
🔹 5.3 Post Check-In Experience
Personalized welcome
Attendee number (gamified)
Speaker showcase
Social follow buttons
Engagement question (interest poll)
🔹 5.4 Admin Dashboard
Total attendees
First-time vs returning
City distribution
Role breakdown
Interest analytics
🔹 5.5 Data Storage
Supabase database
Real-time data tracking
Structured schema for analytics
🧱 6. Tech Stack
Frontend: React / Next.js + Tailwind
Backend: Supabase
Hosting: Vercel / Netlify
QR: Static URL
🔄 7. User Flow
👤 Attendee Flow:

QR Scan → Landing Page → Form → Submit → Experience Screen

🧑‍💻 Admin Flow:

Login → Dashboard → View analytics → Export data

🎨 8. UX Principles

Smooth transitions
